# 🌿 Greenden Tailwind(Frontend Project)

Greenden is a simple and elegant **plant-themed website** built using HTML, CSS, and Tailwind CSS.  
This project showcases a clean UI with structured layouts using modern CSS techniques like Grid and utility-first styling.

## 📌 Overview

The website is designed to represent an online plant store with essential pages and a visually appealing layout.  
It focuses on building responsive sections and improving frontend design skills.

## 📄 Pages Included

- **Home Page** – Includes navigation bar, introduction, about section, best sellers, customer reviews, and newsletter subscription
- **Products Page** – Display of available plants in a grid layout
- **Contact Page** – Basic contact form and details

## 🚀 Features

- Responsive layout using Tailwind CSS
- Grid-based product section
- Clean and modern UI design
- Multi-page navigation
- Utility-first CSS approach

## 🛠️ Built With

- HTML5  
- CSS3  
- Tailwind CSS (for styling and responsiveness)
